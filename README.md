<h2> Installation </h2>
1. NeuronC runs under virtually any Unix system, and is currently developed under Linux
2. <a href="http://vrc.med.upenn.edu/files/nc.tgz"> Download </a> the NeuronC
3. Unzip the folder and install on you system
